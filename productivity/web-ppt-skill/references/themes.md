# 主题色预设（Themes）

17 套精心调配的主题色板，分为叙事轴和展示轴。**只允许从预设中选，不允许自定义 hex 值。**

---

## 叙事轴主题（适合讲故事、有节奏感的内容）

### 1. 🖋 墨水经典（Narrative · 默认）

**适合**：通用分享、商业发布、科技产品、任何场景都安全的默认选择。
**调性**：纯墨黑 + 暖米白，杂志感最强，Monocle / Apricot / A Book Apart 风。

```css
--ink: #0a0a0b;
--ink-rgb: 10, 10, 11;
--paper: #f1efea;
--paper-rgb: 241, 239, 234;
--paper-tint: #e8e5de;
--ink-tint: #18181a;
```

---

### 2. 🌊 靛蓝瓷（Narrative）

**适合**：科技/研究/数据分享、工程师文化、深度内容、技术发布会。
**调性**：深靛蓝 + 瓷白，冷静、理性、有深度，像学术期刊或蓝印花瓷器。

```css
--ink: #0a1f3d;
--ink-rgb: 10, 31, 61;
--paper: #f1f3f5;
--paper-rgb: 241, 243, 245;
--paper-tint: #e4e8ec;
--ink-tint: #152a4a;
```

---

### 3. 🌿 森林墨（Narrative）

**适合**：自然/可持续/文化/非虚构内容、户外品牌、环保主题。
**调性**：深森林绿 + 象牙，沉稳、有呼吸感，像旧版《国家地理》。

```css
--ink: #1a2e1f;
--ink-rgb: 26, 46, 31;
--paper: #f5f1e8;
--paper-rgb: 245, 241, 232;
--paper-tint: #ece7da;
--ink-tint: #253d2c;
```

---

### 4. 🍂 牛皮纸（Narrative）

**适合**：怀旧/人文/阅读/历史/文学分享、独立杂志、手作品牌。
**调性**：深棕 + 暖米，像牛皮信封或老笔记本，温暖、有年代感。

```css
--ink: #2a1e13;
--ink-rgb: 42, 30, 19;
--paper: #eedfc7;
--paper-rgb: 238, 223, 199;
--paper-tint: #e0d0b6;
--ink-tint: #3a2a1d;
```

---

### 5. 🌙 沙丘（Narrative）

**适合**：艺术/设计/创意/时尚分享、画廊手册、审美优先的私享会。
**调性**：炭灰 + 沙色，克制、高级、中性，像沙漠黄昏或建筑设计图册。

```css
--ink: #1f1a14;
--ink-rgb: 31, 26, 20;
--paper: #f0e6d2;
--paper-rgb: 240, 230, 210;
--paper-tint: #e3d7bf;
--ink-tint: #2d2620;
```

---

### 6. Vintage Editorial（Narrative）

**适合**：witty、confident、editorial、personality-driven 内容。
**调性**：几何图形 + 衬线字，文学感强。

**字体**：Fraunces（700/900）+ Work Sans（400/500）

```css

--text-primary: #1a1a1a;
--text-secondary: #555;
--accent-warm: #e8d4c0;
```

---

### 7. Paper & Ink（Narrative）

**适合**：editorial、literary、thoughtful 内容。
**调性**：温暖的米白 + 炭黑 + 猩红点缀，优雅文学感。

```css

--accent-crimson: #c41e3a;
```

## 切换原则

- **一份 deck 只用一套主题**，不要中途换色
- 选定后，在项目记录里备注，方便后续迭代保持一致
- `--ink` / `--paper` / `--accent` 三色体系，确保 CSS 变量命名统一

---

## ❌ 不要做的事

- ❌ **不允许混搭**（例如 ink 取墨水经典的，paper 取沙丘的）
- ❌ **不允许用户自定义 hex 值**——需委婉拒绝并展示 17 套预设让选
- ❌ **不用系统字体**（Arial、Inter、Roboto）—— 用 Fontshare 或 Google Fonts
